## run

1. Clone repo first
   ```bash
   git clone https://github.com/azkadev/specta_compute.git
   cd specta_compute
   ```

2. Run with Docker
   ```bash
   docker build . -t specta_compute
   docker run -it specta_compute
   ```
3. Run with dart
   ```bash
   dart pub get
   dart run
   ```

4. run native exe
    ```bash
    dart pub get
    dart compile exe ./bin/specta_compute.dart -o ./index
    chmod 777 ./index
    ./index
    ```
